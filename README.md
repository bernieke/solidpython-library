This is a library for SolidPython2.

Containing functions to create:
* fillet holes (subtract the resulting object from a corner)
* rounded cubes
* heatset insert holes
* screw holes (optionally countersunk)

The repository also contains some examples on:
* additional SolidPython(2) niceties (BASICS.md)
* how to use openscad-fillets/fillets3d.scad (examples/fillet_library.py)
* how to use threadlib/threadlib.scad (examples/thread_library.py)

To install the library run:
`pip install -e . --break-system-packages`
